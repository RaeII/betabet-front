/speckit-specify Um backend para um app para fazer aposta de jogos da copa do mundo, onde será possivel apostar o placar do jogo, e o usuário pode pontuar se acertar o resultado (time que ganhou ou empate) e o  placar da partida.

- Os usuário podem criar grupos de aposta, podendo colocar nome e imagem e enviar convites a outros usuário e outros usuário podem silicitar a um grupo para entrar e participar
- Os usuários poderam enviar link de acesso de um grupo com código de indicação
- O grupo vai ter um admin (Quem criou) para configurar o grupo e aceitar e remover usuários, um admin pode adcionar outros admin, mas sempre vai ter o principal
- O admin vai pode configurar quantos pontos ganham se acertar o resultado (empate ou equipe vencedora) e o pontos se acertar o placar e outras configuração do grupo, essa opção aparece apenas para o(s) admin(s)
- Cada grupo vai ter um raking com base nos pontos, sempre atualizado
- Os usuário vão pode ver as apostas dos outros usuário apenas quando começar a partida, poderam ver antes da aprtida apenas quem fez as 3 indicações do app (Recurso liberado por recomendação)
- Antes da partida começar e depois vai aparecer um grafico mostrando como está as apostas, mostrando qual seleção estão apostando para vitória, como o grafico do polymarket (Recurso liberado por recomendação)
- Um usuário pode editar sua aposta até 15min antes da partida começar
- Quando fazer uma aposta colocar a opção se replicar a aposta para todos os grupos (apenas se o usuário tiver em mais de um) ou apenas para aquele grupo
- Grupo pode decidir em mostrar as apostas dos usuário da partida antes de começar ou não
- Usuários podem reagir com emojis nas apostas
- organizar os jogos por rodada, grupos e fase, com layout organizado, por exemplo quando for nas chaves, mostras a linhas das chaves com o jogos e podendo clicar na partida e abrir as
  informarções do jogo, mesma coisa do grupo

- Cria um sistema de recomendação, onde o usuário pode enviar link de recomendação e desbloquear recursos por recomendar, tem que conseguir pelo menos 3 indicações para desbloquear

Rotas para admin do sistema:

- Rotas exclusiva para admin,  com verificaão de verificação para admin
- Cadastrar as partidas, as seleções, bandeira e mais informações necessário
- Cadastrar partidas, horário, estádio e outras informações, rodada e grupo
- Ver números do app, quantidade de usuário, grupo, quantidade de aposta, ver a % de aposta por seleção por jogo, isso por grupo e todas as aposta no app
- Entrar nos grupo para ver as informações como se fosse um usuário do grupo mas sem saber
- Ver estatiscas que podem ser interessantes e relevantes a ser analisada
- Ver a quantidade de grupo que um usuário criou
- Ver o número de recomendações
